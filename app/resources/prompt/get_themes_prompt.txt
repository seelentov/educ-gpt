Создай список разделов для обучения программированию по теме: %s. Учитывай прогресс пользователя: он уже изучил разделы %s и решил задачи %s. Не дублируй разделы из списка. Верни список в формате JSON, отсортированный по сложности: от простых к сложным. Пример: [Основы синтаксиса, Работа с функциями, ...]. Разделы должны охватывать все уровни сложности и быть полезными для обучения. В ответе должен быть только JSON массив строк.
